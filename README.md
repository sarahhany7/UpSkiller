## ✨ About TalentAI

**TalentAI** is a stunning, futuristic smart recruitment platform built with PHP + MySQL. It features a modern Dark UI with Glassmorphism design, AI-powered job matching, and real-time notifications.

---

## 🎨 Design Features

- **Dark Modern UI** with Glassmorphism effects
- **Neon Gradients** (Cyan, Purple, Pink)
- **Animated Interactions** & Smooth Transitions
- **Fully Responsive** for Mobile/Tablet/Desktop
- **Premium SaaS-style Dashboard**
